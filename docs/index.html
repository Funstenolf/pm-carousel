PM-Carousel

Full documentation on GitHub

PM-Carousel, for "Pimp My Carousel" is a pretty small, accessible and vanilla JavaScript to manage your carousels. It has no dependencies!

Complete example

const demo1 = document.querySelector("#demo1") pmCarousel({}, demo1);

const demo7 = document.querySelector("#demo7") pmCarousel({ default: { group: 1, }, responsive: [ { minWidth: "400px", group: 2, }, { minWidth: "600px", disable: true, }, { minWidth: "800px", group: 3, autoplay: 4000, }, ] }, demo7);
